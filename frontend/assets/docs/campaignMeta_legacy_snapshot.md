## Overview

Included are the movies that need to be added, whether they be visible from the start or unlocked through a secret method. In addition, the movie sets that go hand in hand with these additions are also included.

This document is only to provide a guideline and basis for what needs to be added and how the systems for each unlock will work.

## Unlockable Movies

While the search option is going to eventually be opened up, the unlocked movies will be included in the pool of movies that are included in the select screen's carousel of movies. In addition, these movies will come preloaded with a second set of specials as an added bonus.

Some movies are already included in the pool. Those ones need to be updated and have their visibility changed.

To unlock, the order of the movies can vary but when the set is unlocked, when it is shown in the movie set carousel in the select screen, you must show the roster of movies in the exact order I have them in. 

For the movie sets that have an "Either" option, have it be randomized everytime the movie set is displayed on the select screen.

For the 'Universal Rewards', they will all stack together. Since they all do something different, the stacking will not affect normal gameplay enough to where the game is broken.

For the details that mention "Team Strike" or "Status Effects" please refer to the built in systems we already have in place.

While there may be some contradiction between this list and the details given above, this list and the details provided down here will be the final source of truth when it comes to the movies provided in each set, what each unlock will be and anything that has to do with the movie sets as a whole. # Files needed to create
Listed are the files that need to be created and/or modified in order to allow this system to work. They should be named as they are in this file and located where described.

## Migrated Files

* campaignMeta.js
    * Includes all unlock rules for hidden movie sets and archetypes
    * moved to matinee folder

## New Files

* extraMovies.js
    * Includes unlock rules for all hidden movies that are not already included in sets/archetypes
* extraDefined.js
    * Source of truth for discoverable sets, ordered display roster, "Either" slot options, campaign-only visibility, and unlock order behavior.
* extraRewards.js
    * Structured reward payloads for:
        * universal rewards
        * active-set-in-run rewards
        * extra-special grants
* bonusSpecials.js
    * The "second set of specials" data for unlocked movies / set bonuses.
    * ALL extra sets for each movie should be moved here
* bonusSelectSets.js
    * Select-screen campaign set carousel adapter:
        * shows only unlocked campaign sets
        * preserves unlock order
        * displays canonical roster order
        * randomizes "Either" display choice each render
    * UI-facing helper for showing unlocked matinee sets in carousel order (with Either randomization on render).
* storage.js
    * Persistence shape for matinee progress in GameState + localStorage (unlocked movies, unlocked sets, counters, streaks, per-run state).
* extraSystems.js
    * Runtime evaluator that reads extraMovies.js, checks GameState stats/run data, and unlocks movies when rules pass.
* extraDiscovery.js
    * Checks current roster/run context against extraDefined.js, discovers sets, and records unlock order.
* extraEffects.js
    * Applies rewards/effects from extraRewards.js at the right lifecycle points (battle start/end, round start, campaign progression).
* bonusSpecialsLoadout.js
    * Injects/merges extra specials from bonusSpecials.js into actor special lists during campaign runs.

# Final Details
These sets and these sets only will only be usable in the Campaign mode. With this, these movies are not usable for the quickplay mode. They will only be shown in the select screen archetype carousel and that is all.

The order that they are unlocked will be the order that they will be shown after the other archetypes. For example, if King of the Hill is found before The Prince of Darkness, then King of the Hill will be shown first in the Carousel then The Prince of Darkness.
